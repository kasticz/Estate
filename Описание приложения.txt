Суть приложения: вёрстка статичного сайта по образцу макета.

Использовано:
HTML 
CSS + SASS
JavaScript
Webpack

Особенности:
1) Полная адаптация сайта для всех разрешений от 320px до 1920 px.

2) Использование JavaScript: a) Валидация формы b) LazyLoad изображений: они загружаются только если пользователь проскроллит достаточно близко , c) Реализация "бургера" на мобильной версии сайта.

3) Вёрстка по системе БЭМ.

4) Вёрстка на основе макета реального сайта.

Макет находится в корневой папке.
